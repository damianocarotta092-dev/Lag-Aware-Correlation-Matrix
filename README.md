# Lag-Aware Correlation Matrix

## Project Overview

Traditional Pearson correlation matrices only measure the relationship between two variables at the exact same time step:

$$
Corr(X_t, Y_t)
$$

While working on a standard correlation matrix, I started questioning one important assumption:

> What if two variables are not correlated at the same instant in time, but become correlated after a temporal delay?

This idea became the starting point of the project.

In many real-world systems, variables do not react immediately:

- financial markets react with delays
- sensor systems respond after propagation time
- environmental systems evolve progressively
- business KPIs often influence each other after several hours, days, or weeks

A standard Pearson correlation matrix may completely miss these delayed dependencies.

The objective of this project is therefore to build a **Lag-Aware Correlation Matrix** capable of:

- scanning multiple lag values
- identifying delayed temporal relationships
- discovering hidden correlations invisible to classical Pearson analysis
- visualizing both correlation strength and optimal lag simultaneously

---

# Project Status

This repository represents the **first experimental phase** of a broader project.

At this stage, the goal is not to claim a new statistical method, but rather to:

- validate the intuition behind lag-aware correlation analysis
- verify whether temporal shifts reveal hidden relationships
- create a working exploratory engine
- establish a foundation for future forecasting and AI integration

The initial results are extremely promising and clearly show that several variables previously considered weakly correlated actually exhibit meaningful delayed dependencies.

---

# Core Idea

Standard Pearson correlation measures:

$$
Corr(X_t, Y_t)
$$

The Lag-Aware approach introduced in this project instead evaluates:

$$
\max_k Corr(X_t, Y_{t+k})
$$

where:

- $k$ = temporal lag
- positive lag means delayed reaction
- negative lag means anticipatory behavior

For every pair of variables, the engine:

1. scans multiple lag values
2. computes correlation at each lag
3. identifies the strongest correlation
4. stores both:
   - correlation strength
   - optimal lag value

## 2. Missing Values Handling

Time series analysis is highly sensitive to missing data.

Missing values were handled using:

- Forward Fill (`ffill`)
- Backward Fill (`bfill`)

This approach preserves temporal continuity while minimizing distortions.

---

## 3. Differencing / Detrending

One of the most important steps of the project was removing spurious correlations generated by trends.

Many time series naturally grow or decrease over time.

Without detrending:

- variables may appear strongly correlated
- even when no real relationship exists

To solve this issue, first-order differencing was applied:

$$
X'_t = X_t - X_{t-1}
$$

This transformation converts the series from:

- absolute levels

into:

- temporal variations

The project therefore analyzes:

> how variables move over time

instead of:

> how large variables are.

This step dramatically improves the interpretability of temporal correlations.

# Standard Pearson Correlation Matrix

After detrending, a standard Pearson correlation matrix was computed as the baseline reference.

Main observations:

- most variable pairs showed correlations close to zero
- only a few immediate relationships remained visible
- the matrix suggested that most variables were independent

Example:

| Variable Pair | Pearson Correlation |
|---|---|
| var1 ↔ var2 | 0.03 |
| var3 ↔ var4 | -0.004 |
| var9 ↔ var10 | 0.03 |

At first glance, these variables appeared unrelated.

# Lag-Aware Correlation Matrix

The custom lag-aware engine then scanned multiple temporal lags from:

$$
-10 \leq lag \leq +10
$$

For each variable pair, the engine identified:

- strongest correlation
- corresponding optimal lag

The resulting matrix produced significantly different insights.

# Main Findings

The project demonstrates several important insights:

## 1. Pearson Correlation Has Temporal Limitations

Standard Pearson correlation may completely miss delayed relationships.

Variables can appear independent even when strong temporal dependencies exist.

---

## 2. Lag Contains Valuable Information

Several relationships emerged only after shifting variables in time.

This suggests that:

- temporal structure itself contains predictive information
- delayed interactions are essential in time series analysis

---

## 3. Detrending Was Essential

Without differencing:

- many spurious correlations would have appeared
- trend-driven relationships would dominate the analysis

Differencing made the correlation analysis significantly more reliable.

---

## 4. Hidden Relationships Became Visible

The lag-aware engine successfully identified:

- delayed positive relationships
- delayed negative relationships
- anticipatory dynamics
- reaction delays between variables

---

# Current Limitations

This project is still exploratory and several important statistical challenges remain:

- multiple testing bias
- spurious maximum correlations
- lack of significance testing
- absence of causality validation

At this stage:

- the engine identifies candidate relationships
- but does not yet prove causality
